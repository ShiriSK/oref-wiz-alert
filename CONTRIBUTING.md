# 🤝 איך לתרום לפרויקט

תודה על הרצון לעזור! הפרויקט הזה נועד להציל חיים — כל תרומה משמעותית.

---

## 💡 רעיונות לשיפורים

- תמיכה בנורות **Govee** ו-**Philips Hue**
- גרסת **Android / iOS**
- תמיכה ב-**Raspberry Pi** (הפעלה 24/7 ללא מחשב)
- **התראות רטט** למכשירים חכמים
- **שפות נוספות** — ערבית, רוסית, אנגלית

---

## 🐛 דיווח על באגים

פתח **Issue** חדש עם:
1. תיאור הבעיה
2. מה ניסית לעשות
3. מה קרה בפועל
4. גרסת Windows ופרטי הנורה

---

## 🔧 שליחת קוד

1. עשה **Fork** לפרויקט
2. צור **Branch** חדש: `git checkout -b feature/שם-הפיצ'ר`
3. בצע שינויים ו-commit
4. שלח **Pull Request**
