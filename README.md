## 🛒 מה צריך לקנות?

### נורת Bluetooth (מומלץ) - LotusLamp
- **מחיר:** 110 ש"ח לשתי נורות (חובה לקנות 2)
- **איפה:** מחסני תאורה
- **דגם:** MELK-OC21 / LotusLamp / Smart Atmosphere Light
- **חיבור:** Bluetooth (לא צריך WiFi!)

### נורת WiFi - WiZ (אופציונלי)
- **מחיר:** כ-80 ש"ח
- **איפה:** חנויות חשמל
- **חיבור:** WiFi ביתי

## 🔵 הגדרת נורת Bluetooth (LotusLamp)

### שלב 1: הדליקו את הנורה
הבריגו את הנורה לשקע והדליקו אותה.

### שלב 2: סריקה
1. באפליקציה, בחרו **Bluetooth (LotusLamp)**
2. לחצו על **סרוק**
3. המתינו לסריקה (10-20 שניות)
4. בחרו את הנורה מהרשימה (מתחילה ב-`BE:`)

### שלב 3: בדיקה
לחצו על **בדוק נורה** - הנורה צריכה להבהב באדום.

### פתרון בעיות Bluetooth:
- ודאו ש-Bluetooth מופעל במחשב
- הנורה חייבת להיות **דולקת** (לא כבויה פיזית)
- נסו לכבות ולהדליק את הנורה
- קרבו את המחשב לנורה

## 📶 הגדרת נורת WiZ (WiFi)

### שלב 1: חברו את הנורה ל-WiFi
השתמשו באפליקציית WiZ הרשמית לחיבור הנורה לרשת הביתית.

### שלב 2: סריקה
1. באפליקציה, בחרו **WiZ (WiFi)**
2. לחצו על **סרוק**
3. בחרו את כתובת ה-IP של הנורה

## ⚠️ מגבלות ידועות (Known Limitations)

### 1. נורת Bluetooth לא נכבית לגמרי
**בעיה:** לפעמים הנורה נשארת דולקת אחרי שההתרעה נגמרת.

**פתרון:** האפליקציה שולחת פקודת כיבוי נוספת אחרי דקה. אם עדיין דולקת - כבו והדליקו אותה פיזית.

### 2. נורה כבויה לא מקבלת פקודות
**בעיה:** אם הנורה כבויה לגמרי (שחור), היא לא יכולה לקבל פקודות Bluetooth.

**פתרון:** הנורה צריכה להיות דולקת (גם על עמעום מינימלי) כדי לקבל פקודות.

### 3. טווח Bluetooth מוגבל
**בעיה:** Bluetooth עובד עד כ-10 מטר.

**פתרון:** ודאו שהמחשב קרוב לנורה, או השתמשו בנורת WiFi.

### 4. התרעות נשארות ב-API רק 1-2 שניות
**בעיה:** פיקוד העורף מוחק התרעות מה-API מהר מאוד.

**פתרון:** האפליקציה בודקת כל 0.5 שניות כדי לתפוס התרעות בזמן.

### 5. צריך מחשב דולק
**בעיה:** האפליקציה רצה על המחשב, אז המחשב חייב להיות דולק.

**פתרון:** השאירו מחשב ישן/לפטופ דולק 24/7, או השתמשו ב-Raspberry Pi.

## 🔧 למתקדמים: הרצה עם Python

אם אתה מעדיף להריץ את קוד המקור ישירות:

### שלב 1: התקנת Python
1. הורידו Python מ-[python.org](https://www.python.org/downloads/)
2. **חשוב:** סמנו ✅ "Add Python to PATH" בהתקנה

### שלב 2: התקנת ספריות
פתחו PowerShell והריצו:
```powershell
pip install requests bleak
```

לנורות WiZ הוסיפו:
```powershell
pip install pywizlight
```

### שלב 3: הפעלה
```powershell
python oref_wiz_gui_v3.py
```
