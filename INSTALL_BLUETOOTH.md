# 🔵 מדריך התקנה - נורת Bluetooth (LotusLamp)

## 🛒 שלב 1: קניית הנורה

### איפה לקנות?
- **מחסני תאורה** - 110 ש"ח לשתי נורות
- מגיע בקופסה של 2 נורות (חובה לקנות 2)

### איזה דגם?
חפשו אחד מהשמות הבאים:
- LotusLamp
- MELK-OC21
- Smart Atmosphere Light
- נורה חכמה Bluetooth RGB

### מה לוודא?
- ✅ חיבור Bluetooth (לא WiFi בלבד)
- ✅ תמיכה ב-RGB (צבעים)
- ✅ בית נורה E27 (סטנדרטי)

---

## 💻 שלב 2: התקנת התוכנה

### 2.1 התקנת Python
1. גלשו ל-[python.org/downloads](https://www.python.org/downloads/)
2. הורידו את הגרסה האחרונה
3. **חשוב מאוד:** סמנו ✅ "Add Python to PATH"
4. לחצו Install

### 2.2 התקנת ספריות
פתחו PowerShell (חיפוש → PowerShell) והריצו:

```powershell
pip install requests bleak
```

### 2.3 הורדת האפליקציה
הורידו את `oref_wiz_gui_v3.py` מהריפו.

---

## 🔌 שלב 3: חיבור הנורה

### 3.1 הכנה פיזית
1. הבריגו את הנורה לשקע/מנורה
2. הדליקו את המתג - הנורה צריכה להידלק

### 3.2 הפעלת Bluetooth במחשב
1. הגדרות → Bluetooth והתקנים
2. ודאו ש-Bluetooth **מופעל**

### 3.3 סריקה באפליקציה
1. הפעילו את האפליקציה:
   ```powershell
   python oref_wiz_gui_v3.py
   ```
2. בחרו **Bluetooth (LotusLamp)**
3. לחצו על **סרוק**
4. המתינו 10-20 שניות
5. בחרו את הנורה מהרשימה

### איך מזהים את הנורה?
- הכתובת מתחילה ב-`BE:16:5C:` (בדרך כלל)
- השם יכול להיות `MELK` או `LotusLamp` או `LEDnet`

---

## ✅ שלב 4: בדיקה

1. לחצו על **בדוק נורה**
2. הנורה צריכה להבהב באדום
3. אם עובד - מוכנים!

---

## 🚀 שלב 5: הפעלה

1. בחרו את העיר שלכם
2. לחצו **התחל מעקב**
3. השאירו את האפליקציה רצה ברקע

---

## 🔧 פתרון בעיות

### הנורה לא מופיעה בסריקה
- ודאו שהנורה **דולקת** (לא כבויה פיזית)
- ודאו ש-Bluetooth מופעל במחשב
- נסו לכבות ולהדליק את הנורה
- קרבו את המחשב לנורה (עד 10 מטר)

### הנורה לא מגיבה לבדיקה
- נסו שוב - לפעמים לוקח כמה שניות
- כבו והדליקו את הנורה פיזית
- סרקו מחדש

### הנורה נשארת דולקת אחרי התרעה
- זה באג ידוע בחומרה
- האפליקציה שולחת כיבוי נוסף אחרי דקה
- אם עדיין דולקת - כבו והדליקו פיזית

### שגיאת "bleak not found"
הריצו שוב:
```powershell
pip install bleak
```
